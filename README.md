# 直播字幕小工具

用于直播的字幕小工具，以类似RPG游戏的对话框形式进行提词，同时可以发送到直播间弹幕（B站）

## 运行效果

实现了淡入淡出和打字机效果

![gif录像](./media/screen_shot.gif)

设置页面可以设置房间号，以及登录登出

![截图](./media/screen_shot.jpg)
